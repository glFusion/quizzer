glFusion Quizzer Plugin Configuration

Question
Enter the question to be asked.
Question Type
Select the type of question: either single-answer radio button or multi-answer checkboxes.
Answers
Enter up to 5 possible answers for the question. Check the radio button or checkboxes to indicate the correct answer(s).
Post-Answer Message
Enter an optional message to be displayed immediately after the answer is submitted. This may be a description of the answer explaining why it it the correct one.
Enabled?
Uncheck this box to disable a question. The question will remain in the database for future use but will not be shown on any quizzes until re-enabled.
Randomize Answers?
Check this box to display the answers in a random order. This should be unchecked if you include answers such as "All of the above"
Partial Credit Allowed?
Check to grant partial credit for correctly-answered checkbox questions. Does not apply to radio-button questions.
