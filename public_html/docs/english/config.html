glFusion Quizzer Plugin Configuration

Main Quizzer Options

Make Centerblock?
Select "Yes" to have the first-available quiz displayed as the home page.
Default User Group
Select the default group to be given permission to fill out new forms. This may be overridden during or after creation.
Purge Null Results
Enter the number of days after which result sets containing no answers are purged. Enter zero to disable purging.
