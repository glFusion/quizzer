Forms Plugin

The Forms Plugin allows you to create custom forms to collect information from site users. This can be used for things like signup forms, polls, etc.

Forms Plugin Configuration Options

Main Forms Settings
Option Description
Display glFusion Blocks Select whether to display left blocks, right blocks, both, or neither when a form is displayed. This only affects forms displayed directly via url (site_url/forms/?frm_id=XX). Forms displayed via autotags, which is the recommended method, will not be affected by this setting.
Default User Group Select the default group to be given permission to fill out new forms. This may be overridden during or after creation.
Default Results Group Select the default group to be given permission to view and download the results for new forms. This may be overridden during or after creation.
Field Defaults
Option Description
Default Text Field Size Default size parameter for text fields.
Default "maxlen" for Text Fields Default maximum entry length for text fields.
Default textarea "rows" value Default number of rows for textarea fields.
Default textarea "cols" value Default number of columns for textarea fields.
